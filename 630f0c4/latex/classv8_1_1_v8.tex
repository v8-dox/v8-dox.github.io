Resume execution capability in the given isolate, whose execution was previously forcefully terminated using \hyperlink{classv8_1_1_v8_a8ff3cfb774c8540cf7b2c3967d5755dd}{Terminate\+Execution()}.

When execution is forcefully terminated using \hyperlink{classv8_1_1_v8_a8ff3cfb774c8540cf7b2c3967d5755dd}{Terminate\+Execution()}, the isolate can not resume execution until all Java\+Script frames have propagated the uncatchable exception which is generated. This method allows the program embedding the engine to handle the termination event and resume execution capability, even if Java\+Script frames remain on the stack.

This method can be used by any thread even if that thread has not acquired the \hyperlink{classv8_1_1_v8}{V8} lock with a \hyperlink{classv8_1_1_locker}{Locker} object.
