\subsection{Detailed Description}
\subsubsection*{template$<$typename K, typename V$>$class v8\+::\+Strong\+Map\+Traits$<$ K, V $>$}

A default trait implementation for \hyperlink{classv8_1_1_persistent_value_map}{Persistent\+Value\+Map}, which inherits a std\+:map backing map from \hyperlink{classv8_1_1_std_map_traits}{Std\+Map\+Traits} and holds non-\/weak persistent objects.

Users have to implement their own dispose trait.
