ScriptOrigin *  origin = NULL,

static

Compiles the specified script. The ScriptOrigin* and ScriptData* parameters are owned by the caller of Script::Compile. No references to these objects are kept after compilation finishes.

The script object returned is bound to the context that was active when this function was called. When run it will always use this context.

Local<Value> v8::Script::Run ( )

Runs the script returning the resulting value. If the script is context independent (created using ::New) it will be run in the currently entered context. If it is context specific (created using ::Compile) it will be run in the context in which it was compiled.
